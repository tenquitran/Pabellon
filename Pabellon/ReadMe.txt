Pabellon  - viewer for the Pabellon Barcelona 3D model from eMirage 
(http://www.emirage.org/2013/04/24/free-download-archviz-project-pabellon-barcelona-3d-scene-v1-2-updated/).

Model authors:
additional modeling / shaders / lighting / rendering by Hamza Cheggour
Base model/textures by Claudio Andres
architecture designed by Ludwig Mies van der Rohe

The project uses the following libraries:
	GLEW, version 2.0.0 (http://glew.sourceforge.net/)
	GLM, version 0.9.8.4 (http://glm.g-truc.net/)
	Assimp, version 4.0.1 (http://www.assimp.org/index.html)
	SOIL (http://www.lonesock.net/soil.html)
